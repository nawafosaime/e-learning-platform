# Project Wiki - E-Learning Platform

## Overview
This project is a learning website where doctors create courses, lessons, and quizzes, while students enroll, learn, take quizzes, and track progress.

## Why this project?
We selected this project because it is useful, practical, and suitable for a real educational scenario. It also allowed us to combine frontend, backend, authentication, database design, and teamwork.

## Underlying Technologies
- Semantic HTML
- CSS and Bootstrap
- Raw JavaScript with ES6 classes
- Node.js and Express
- MongoDB and Mongoose

## Setup
1. Install Node.js
2. Install project packages with `npm install`
3. Add `.env` file from `.env.example`
4. Start the project with `npm run dev`

## Flow
Doctor creates course → Doctor adds lessons/quizzes → Student enrolls → Student studies lessons → Student takes quiz → System tracks progress
